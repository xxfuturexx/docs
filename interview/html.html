# HTML面试题

# 01、行内元素有哪些？块级元素有哪些？ 空(void)元素有那些？

行内元素:span、img、input...  ==》不独占一行，并且不能设置宽高
块级元素：div、footer、header、section、p、h1-h6... ==》独占一行，可以设置宽高
空元素:br、hr...

通过css 元素之间的转换问题：
display:inline			把某元素转换成行内元素				==》不独占一行，并且不能设置宽高
display:inline-block	把某元素转换成了行内块级元素		  ==》不独占一行，可以设置宽高
display:block			把某元素转换成块级元素				==》独占一行，可以设置宽高

# 面试题：页面导入样式时，使用link和@import有什么区别？

//01.html 
// link在html文件引入
<link rel="stylesheet" href="./a.css">

// a.css  
// @import在css文件引入
@import './b.css';
div{
    height: 100px;
    background-color: skyblue;
}

// b.css
p{
    background-color: yellow;
}

区别一:link先有，后有的@import(兼容性link比@import好，link兼容ie5)
区别二：加载顺序差别，浏览器先加载的标签link，后加载@import

# 面试题：title与h1的区别、b与strong的区别、i与em的区别？

title与h1的区别：
定义：
	title：概括了网站信息，可以告诉搜索引擎或者用户关于这个网站的主要内容是什么（有利于SEO）
	h1:文章主题内容，告诉蜘蛛我们的网站内容最主要是什么
区别：
	title它是显示在网页标题上，h1是显示在网页内容上
	title比h1更加的重要( title > h1) ==> 对于seo来说
场景:
	网站的logo都是用h1标签包裹的（搜狐新闻网站内容的新闻小标题也是用h1标签包裹的）

b与strong的区别:
都是加粗
定义：
	b：实体标签、用来给文字加粗的
	strong:逻辑标签，用来加强字符语气的
区别：
	b标签只有加粗的样式，没有实际含义
	strong表示标签内字符比较重要、用以强调的
题外话：为了符合css3规范，b尽量少用用strong就行了

i与em的区别：
都是文字倾斜
定义：
	i:实体标签，用来做文字倾斜的
	em:是逻辑标签，用来强调文字内容的
区别：
	i只是一个倾斜标签，没有实际含义
	em表示标签内字符重要，用以强调
场景：
 	i更多的用在字体图标上，em用在专业术语上（医药、生物等）

区别一：

# 面试题：png、jpg、gif 这些图片格式解释一下，分别什么时候用？

png:无损压缩，尺寸体积要比jpg/jpeg的大，适合做小图标
jpg:采用压缩算法，有一点失真，比png体积要小，适合做中大图片
gif:一般是做动图的
webp:同时支持有损和无损压缩，相同质量的图片，webp具有更小的体积。兼容性不是很好（但是还行）

# 面试题：DOCTYPE标签 doctype（声明、文档类型）

定义和用法:
<!DOCTYPE> 声明必须是 HTML 文档的第一行，位于 <html> 标签之前。

<!DOCTYPE> 声明不是 HTML 标签；它是指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。

在 HTML 4.01 中，<!DOCTYPE> 声明引用 DTD，因为 HTML 4.01 基于 SGML。DTD 规定了标记语言的规则，这样浏览器才能正确地呈现内容。

HTML5 不基于 SGML，所以不需要引用 DTD。

# 面试题:meta标签

https://blog.csdn.net/m0_54249336/article/details/113771124

一. 什么是meta标签？meta(元语言，元宇宙)

  • 元数据就是指非显示型数据，机器可读型数据
  • meta 标签提供各种元数据，从而提供各种文档信息

二. meta标签如何使用

在head标签中可以添加多个meta标签，meta标签有四大属性：

  • charset属性： 用于声明页面的字符集编码，一定要写在第一行，否则将会出现乱码

    <meta charset="utf-8">

  • http-equiv属性： 定义这个属性会添加到http头部，可用与刷新重定向和设置Cookie等操作，与content属性结合使用

    //不想用JS来进行自动刷新重定向，可以直接用meta标签实现
//令http-equiv为refresh, 在content中可以定义刷新时间和URL
<meta http-equiv="Refresh" content="5;url=http://blog.yangchen123h.cn" />
<meta http-equiv="Set-Cookie" content="cookievalue=xxx; expires=Friday, 12-Jan-2001 18:18:18 GMT； path=/">

  • name属性：常用于设置网站关键字和描述方便SEO，也常用于通过viewport（视口）调整页面的展示方式，要与content属性结合使用

    <!-- 设置网站关键字，方便SEO -->
<meta name="keywords" content="供应德州土工膜、德州土工膜供应，德州土工膜批发市场，德州土工膜批发，德州土工膜行情，德州土工膜">
<!-- 是否启用 WebApp 全屏模式 -->
<meta   name="viewport"  content="viewport-fit=cover, width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no"/>
